## quick start

1. `$ git clone https://github.com/mjhea0/node-base.git your_app`
2. `$ cd your_app`
3. `$ npm install`
4. `$ node app`

## test

1. `$ make test`
2. `$ make coverage`


## config (nconf)

1. edit config files in "/config" directory

## logging (winston)

1. add logging settings to config files

## seed database

1. `$ node seed.js`
